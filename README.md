# MealFlix 

MealFlix is a modern, Netflix-inspired web app that lets users discover and explore recipes in a visually engaging way. It uses a clean UI with smooth animations and interactive meal browsing.


##  Live Demo
 https://richhyyy.github.io/mealflix/


##  Features

-  Search meals by name
-  Random meal generator
-  Netflix-style horizontal scrolling UI
-  Hover preview with meal titles
-  Step-by-step recipe instructions
-  Ingredients list with measurements
-  Smooth animations and transitions


##  Tech Stack

- HTML
- CSS (Flexbox, animations)
- JavaScript (Fetch API)
- TheMealDB API


##  What I Learned

- Working with APIs using Fetch
- DOM manipulation in JavaScript
- UI/UX design improvements
- Creating smooth animations
- Structuring a frontend project
